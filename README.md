# Project-Uni

Project Uni is a community-built Synthetic Intelligence ecosystem and public website prototype.

## Website Architecture

- [Project Uni Website Blueprint](WEBSITE_BLUEPRINT.md) is the master specification for the website experience before implementation begins.
- The website direction is an interactive chapter-based digital experience, not a generic startup landing page.

## Current Website Artifacts

- `Project Uni.html` is the current Project Atlas / Project Uni digital headquarters page.
- `webpage` appears to mirror the main public website HTML export.
- `Project Atlas corrected.html` contains verification notes from an earlier corrected build artifact.

## Build Sequence

1. Approve the website blueprint.
2. Create the UI/UX design system.
3. Write the final story and copy.
4. Implement the approved experience.
5. Polish, test, and launch.
